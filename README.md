# CRM AI Agent 🤖

An AI-powered CRM assistant that can:

## Features
- Create leads using natural language
- Update lead status
- Smart lead search (name, email, company)
- Generate quotations from leads
- Hybrid extraction (Regex + LLM)

## Tech Stack
- Python
- ERPNext API
- Ollama (TinyLlama)
- Regex + LLM Hybrid NLP
